Introduction

CSS, or Cascading Style Sheets, is a markup language used for styling and formatting web pages. With CSS, you can control the appearance of text, images, and other elements on your website. This technical documentation will cover the basic syntax, selectors, properties, and values of CSS.

Basic Syntax

CSS rules are comprised of a selector and a declaration block. The selector indicates which HTML elements the rule applies to, and the declaration block contains one or more property-value pairs separated by a semicolon.

selector { property: value; property: value; property: value; }

Here's an example of a CSS rule that sets the background color of all paragraphs to yellow:

p { background-color: yellow; }
Selectors

CSS selectors are used to target specific HTML elements. There are several types of selectors:

Element Selector
Targets HTML elements based on their tag name. For example, p targets all p elements.

p { /* properties and values */ }
Class Selector

Targets HTML elements with a specific class name. The class name is preceded by a period. For example, .my-class targets all elements with class="my-class".

.my-class { /* properties and values */ }
ID Selector

Targets HTML elements with a specific ID. The ID is preceded by a hash symbol. For example, #my-id targets the element with id="my-id".

#my-id { /* properties and values */ }
Attribute Selector

Targets HTML elements with a specific attribute. The attribute is enclosed in square brackets. For example, [href] targets all elements with an href attribute.

[href] { /* properties and values */ }
Descendant Selector

Targets HTML elements that are descendants of another element. The descendant selector uses a space to separate the parent and child elements. For example, ul li targets all li elements that are descendants of a ul element.

ul li { /* properties and values */ }
Properties and Values
CSS properties are used to specify the visual style of HTML elements. Properties are followed by a colon, and values are followed by a semicolon. Some common properties and values include:
Color
Specifies the color of text or background. Values can be named colors, hexadecimal values, or RGB values. color: red; background-color: #000000;
Font
Specifies the font family, size, weight, and style of text.
Types of fonts :
  • Arial
  • Times New Roman
  • Helvetica
  • Georgia
  • Open Sans
  • And more ...
font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; font-style: italic;
